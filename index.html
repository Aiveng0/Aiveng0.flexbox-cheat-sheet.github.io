Поперечный конец Flex-контейнер

Свойства flex-контейнера

1

display: flex;

display: -webkit-flex;

Назначение: запуск механизма Flexbox

Возможные значения: flex / inline-flex

flex;

inline-flex;

2

flex-direction: row;

-webkit-flex-direction: row;

Назначение: направление укладки элементов по осям

Возможные значения: row / row-reverse / column / column-reverse

row;

column;

row-reverse;

column-reverse;

3

flex-wrap: nowrap;

-webkit-flex-wrap: nowrap;

Назначение: расположение элементов в одну или несколько строк

Возможные значения: nowrap / wrap / wrap-reverse

nowrap;

wrap;

wrap-reverse;

4

flex-flow: row nowrap;

-webkit-flex-flow: row nowrap;

Назначение: сокращённая запись для свойств flex-direction и flex-wrap

Возможные значения: см. свойства 2 и 3

5

justify-content: flex-start;

-webkit-justify-content: flex-start;

Назначение: выравнивание flex-элементов по главной оси текущей строки

Возможные значения: flex-start / flex-end / center / space-between / space-around

flex-start;

center;

space-around;

flex-end;

space-between;

6

align-items: stretch;

-webkit-align-items: stretch;

Назначение: выравнивание flex-элементов вдоль поперечной оси

Возможные значения: stretch / flex-start / flex-end / center / baseline

stretch;

flex-end;

baseline;

flex-start;

center;

7

align-content: stretch;

-webkit-align-content: stretch;

Назначение: выравнивание строк flex-элементов вдоль поперечной оси

Возможные значения: stretch / flex-start / flex-end / center / space-between / space-around

stretch;

flex-end;

space-between;

flex-start;

center;

space-around;
